# sysspeaker

## Installation

Follow the steps below to get started with sysspeaker.

### 1. Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install sysspeaker.

```bash
pip install sysspeaker
```

### 2. Importing

```bash
import sysspeaker
```

### 3. Usage

```bash
sysspeaker.speak("Your message here")
```
